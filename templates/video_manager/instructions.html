Инструкции по добавлению видео в AR приложение

Важно! Все добавленные видео автоматически синхронизируются с Firebase и появляются в Unity приложении ComeBack.
Пошаговое руководство:
Шаг 1: Получение GPS координат
  1. Перейдите на сайт LatLong.net
  2. В поле поиска введите адрес или название места
  3. Нажмите "Find" или кликните на карте
  4. Скопируйте значения Latitude и Longitude
Примеры адресов:
• Tashkent, Uzbekistan
• Amir Temur Square, Tashkent
• 41.2995, 69.2401
Шаг 2: Подготовка видео
Требования к видео:
  • Формат: Только MP4 H.264
  • Разрешение: 1920x1080 или меньше
  • Размер: до 5 МБ
  • Длительность: до 3 минут
  • Хромакей: #00b23f (зеленый фон для прозрачности)
Firebase Storage

Видео автоматически загружается в Firebase Storage и становится доступным для Unity приложения в реальном времени.

Хромакей (зеленый экран) - #00b23f
#00b23f
RGB: 0, 178, 63
Правила съемки с хромакеем:
  • Фон: Используйте зеленый экран точно цвета #00b23f
  • Освещение: Равномерно осветите зеленый фон, избегайте теней
  • Одежда: НЕ используйте зеленые оттенки в одежде и аксессуарах
  • Расстояние: Держите объект на расстоянии 1-2 метра от фона
  • Качество: Избегайте складок и неровностей на зеленом фоне
Важно: Unity автоматически сделает цвет #00b23f прозрачным в AR пространстве.
Процесс добавления видео
1. Найти координаты

Используйте LatLong.net для получения точных GPS координат места

2. Подготовить видео

Загрузите файл или подготовьте ссылку на видео

3. Сохранить

Заполните форму и сохраните - видео появится в приложении

Как пользоваться LatLong.net
Получение координат места:
  1. Откройте {{ latlong_url }}
  2. В поле "Place Name" введите адрес на английском языке
  3. Нажмите кнопку "Find" или Enter
  4. Координаты появятся в полях "Latitude" и "Longitude"
  5. Скопируйте эти значения в форму добавления видео
Совет: Для более точных результатов добавляйте код страны, например: "Tashkent, UZ" вместо просто "Tashkent"
Примеры координат:
Ташкент: 41.2995, 69.2401
Самарканд: 39.6542, 66.9597
Бухара: 39.7747, 64.4286
Технические детали
Требования к видео:
  • Формат: Только MP4 H.264
  • Размер файла: до 5 МБ
  • Длительность: до 3 минут
  • Разрешение: до 1920x1080
  • Хромакей: #00b23f
Кодирование:
  • Кодек видео: H.264
  • Профиль: High/Main
  • Битрейт: 3-8 Mbps
  • FPS: 24-30
Как работает система:
  1. Видео загружается в Firebase Storage
  2. Данные сохраняются в Firebase Realtime Database
  3. Unity получает обновления мгновенно
  4. AR объект появляется по GPS координатам
  5. Хромакей автоматически удаляется в Unity
Время появления в приложении: мгновенно после загрузки
